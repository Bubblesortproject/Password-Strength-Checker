# Password-Strength Checker
A Python based cybersecurity tool that evaluates the strength of user passwords by analyzing multiple security factors including password length, uppercase and lowercase letters, numbers, and special characters. The program rates password strength and provides actionable feedback to help users create more secure passwords.
Features
Password strength analysis (Weak / Moderate / Strong / Very Strong)
Real-time security improvement suggestions
Secure random password/passkey generator
Simulated biometric authentication options
Technologies Used
Python
re (Regular Expressions)
random
string
